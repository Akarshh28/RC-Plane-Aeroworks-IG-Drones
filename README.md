# RC Plane Development – Aeroworks 4.0

## Workshop Details
Participated in a 4-day hands-on aerospace workshop "Aeroworks 4.0", sponsored by IG Drones and organized by The Drone Learner’s Club.

Worked in a 6-member team under senior mentorship to design and assemble a Remote-Controlled (RC) aircraft.

---

## Project Objective
- Understand aircraft structure and subsystem integration
- Study propulsion and control systems
- Learn basics of aerodynamics
- Perform flight simulation
- Observe real-world flight testing

---

## Aircraft Architecture Overview

### 1. Propulsion System
- BLDC Motor
- Electronic Speed Controller (ESC)
- Li-Po Battery
- Propeller

### 2. Control System
- Transmitter & Receiver
- Servo Motors
- Aileron (Roll Control)
- Elevator (Pitch Control)
- Rudder (Yaw Control)

---

## Technical Learning

### Aerodynamics
- Lift generation via airfoil structure
- Thrust vs Drag
- Center of Gravity importance

### System Integration
- Wiring ESC to BLDC motor
- Signal control via PWM
- Servo calibration basics

### Flight Simulation
- Practiced joystick-based simulation
- Understood aircraft stability & control response

---

## My Contribution
- Assisted in airframe assembly
- Participated in component mounting
- Observed signal calibration
- Took part in flight simulation exercises

---

## Tools & Components Used
BLDC Motor, ESC, Li-Po Battery, Servo Motors, RC Transmitter, Foam Airframe, Propeller
